# Qt dashboard

## Example

- Qt dashboard is a example of Qt.

- Qt\Examples\Qt-5.13.0\quickcontrols\extras\dashboard

## How to test

1. Using Qt Creator
	1. execute qt-creator
	2. load 'dashboard.pro'
	3. compile and execute

2. Using QML Scene
	1. Start Qt Command
	2. cd qt-dashboard\dashboard\qml
	3. execute command
		- qmlscene dashboard.qml
